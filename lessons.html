Designing - Mockup

Mockups are high fidelity representations of websites, showing detailed elements like layout, color schemes, typography, and interactive features. I learnt how to make mockups in Term 2 of my Year 11 in MAGS. Mr.Robins started teaching it to us by first making hand-drawn mockups.

After that, we used the software "Figma" to make high-feidility digital mockups in our laptops. We learnt the Nielsen's 10 Usability heuristics and their applications in websites. Keeping those in minds, we created I along my classmates created our own Mockups.

Language - html & css

html is a coding laguage that is used to make website structures. We were taught the basic of html and the tags that are present and their functions. We learnt the importance of indentations in codes and how to make it easier to navigate through our codes as well as the conventions of folders for coding html and css.


This is the first practise test that we did where we were told to make a nvaigation bar using the tags we had already learnt. The focus was whether we could make a nvigation bar with menu buttons that are in a row and there is a slight gap between the side of the web and the first nvaigation button.

html and css together are very essential when making a website. html gives the website structure and determines what will be in it while css makes the website more appeasing providing the ability to customize the structure and easy to navigate.
